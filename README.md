# RequestTool

## 介绍

- 此仓库是 `IDEA` 插件 `RequestTool` 的反馈目录和教程目录
- `RequestTool` 是一个简单调试的 Http 工具.
- 其构建的主要目的，是为了集成到IDE里面。
- 其次是为给其他插件提供支持
